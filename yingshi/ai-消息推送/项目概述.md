# 项目概述

## 项目背景

- RSSAnt（蚁阅）开源项目的扩展版本，结合 AI 报告、飞书机器人等业务能力，支持本地与容器化部署。

## 项目目标

- 提供稳定的 RSS 订阅与全文阅读能力，支持跨平台阅读、收藏与已读管理。
- 集成 AI 行业搜索与报告生成、飞书机器人定时推送等增值功能。

## 功能概述

- 订阅源管理、文章读取与状态管理（收藏、已读）。
- 行业资讯聚合（Hacker News/GitHub/ArXiv）与 AI 内容加工。
- AI 娱乐与 AIGC 报告生成与导出（例如导出到飞书文档）。
- 飞书机器人配置管理与定时推送。
- 前后端分离，后端提供 REST API 与 Swagger 文档。

## 技术栈

- 后端：Django、DRF、AllAuth、Whitenoise、PostgreSQL、supervisor。
- 前端：React 18、TypeScript、Vite、Tailwind、Radix UI/shadcn、Zustand。
- 调度与多角色：`api/worker/scheduler` 多角色运行，`aiohttp` 服务（rssant_scheduler）。
- 容器与部署：Docker 多阶段构建、supervisord 管理进程、脚本化部署。

## 架构类型

- 未使用 GBF 框架；后端采用 Django + DRF 的分层组织，领域模型以 Django ORM 模型为核心，配合 `UnionFeed/UnionStory` 聚合访问层与服务层。
