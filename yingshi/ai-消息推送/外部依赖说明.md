# 外部依赖说明

## 下游服务与外部系统

- 数据库：PostgreSQL 11（容器中安装，初始化脚本 `box/initdb.sql`）。
- Web 服务器：Nginx（容器内，静态与前端构建产物托管）。
- Cloudflare Worker（可选）：RSS 代理（`cloudflare_worker/rssant/README.md:1`）。
- RSSHub（可选）：公共或自部署实例（示例变量见 `box/rssant.env:33`）。
- GitHub OAuth：`allauth.socialaccount` 与配置（`rssant/settings/settings.py:218`）。
- SMTP 邮件（可选）：通知与注册邮件（`rssant/settings/settings.py:228`）。
- 飞书开放平台：飞书文档导出与机器人推送（`rssant_api/views/story.py:336`、`rssant_api/views/feishu_bot.py:35`）。

## 调用关系图（概览）


## 配置与环境变量

- 示例参考：`box/rssant.env:1`。
- 重要变量：`RSSANT_SECRET_KEY`、`RSSANT_ROOT_URL`、`RSSANT_CHECK_FEED_MINUTES`、`RSSANT_PG_*`、`RSSANT_ROLE`、`RSSANT_BIND_ADDRESS`。
- 可选变量：RSS 代理与 RSSHub、SMTP、GitHub OAuth、飞书 App ID/Secret。
- 安全要求：真实密钥与凭据不得提交到仓库。
